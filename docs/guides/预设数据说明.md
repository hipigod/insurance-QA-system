# 预设数据说明文档

## 📋 数据概览

系统已成功初始化以下预设数据:

### 👥 客户角色 (4个)

#### 1. 小白客户
- **难度**: 简单
- **特点**: 对保险了解不多，需要耐心解释
- **适用场景**: 新手销售人员练习基础沟通技巧
- **关键异议**:
  - "保险不就是骗人的吗?"
  - "这个保费太贵了，我负担不起"
  - "我回家和爱人商量一下"

#### 2. 理性分析型
- **难度**: 中等
- **特点**: 注重数据对比，需要详细说明
- **适用场景**: 练习专业产品知识和数据展示
- **关键异议**:
  - "这个病为什么不保?"
  - "XX公司的产品比你们好"
  - "你说的这些真的能兑现吗?"

#### 3. 价格敏感型
- **难度**: 困难
- **特点**: 预算有限，需要高性价比产品
- **适用场景**: 练习价值塑造和价格谈判
- **关键异议**:
  - "太贵了，我买不起"
  - "这么贵能保障什么?"
  - "等我有钱了再买"

#### 4. 怀疑抗拒型
- **难度**: 困难
- **特点**: 对保险行业有偏见，需要建立信任
- **适用场景**: 练习信任建立和异议处理
- **关键异议**:
  - "保险公司都是骗子"
  - "到时候肯定不赔钱"
  - "我之前被骗过"

---

### 🏦 保险产品 (3个)

#### 1. 少儿教育金保险
- **类型**: 教育金
- **保费范围**: 5000-20000元/年
- **目标人群**: 0-12岁儿童的父母
- **核心卖点**:
  - 专款专用，确保孩子教育资金
  - 保费豁免功能
  - 灵活领取，覆盖教育各阶段
  - 复利增值

#### 2. 终身重大疾病保险
- **类型**: 重疾险
- **保费范围**: 3000-15000元/年
- **目标人群**: 18-55岁健康人群
- **核心卖点**:
  - 覆盖100+种重疾、20+种中症、40+种轻症
  - 确诊即赔，不限制用途
  - 终身保障
  - 多次赔付(可选)

#### 3. 定期寿险
- **类型**: 寿险
- **保费范围**: 500-3000元/年
- **目标人群**: 20-50岁家庭经济支柱
- **核心卖点**:
  - 低保费高保障(几百元撬动百万保额)
  - 纯保障，无理财成分
  - 期限灵活
  - 免责条款极少

### 📊 评分维度 (4个)

#### 1. 需求挖掘 (权重: 20%)
评估销售人员是否准确了解客户真实需求和担忧

**评分标准**:
- 提问质量 (25分)
- 倾听能力 (25分)
- 需求确认 (25分)
- 针对性推荐 (25分)

#### 2. 产品说明 (权重: 25%)
评估产品介绍是否准确、清晰、易懂

**评分标准**:
- 准确性 (30分)
- 清晰度 (30分)
- 针对性 (20分)
- 证据支持 (20分)

#### 3. 异议处理 (权重: 25%)
评估是否有效化解客户疑虑和反对意见

**评分标准**:
- 情绪管理 (25分)
- 异议识别 (25分)
- 处理技巧 (30分)
- 效果评估 (20分)

#### 4. 促成能力 (权重: 30%)
评估是否能有效引导客户完成购买决策

**评分标准**:
- 时机把握 (25分)
- 促成技巧 (35分)
- 临门一脚 (25分)
- 专业性 (15分)

## 🎯 使用建议

### 新手训练路径
1. **第1周**: 小白客户 + 少儿教育金
   - 练习基础沟通和产品介绍

2. **第2周**: 理性分析型 + 终身重疾险
   - 练习专业知识和数据展示

3. **第3周**: 价格敏感型 + 定期寿险
   - 练习价值塑造和价格谈判

4. **第4周**: 怀疑抗拒型 + 任一产品
   - 综合练习高难度客户应对

### 评分重点分布
根据权重分配,销售人员应重点关注:
1. **促成能力** (30%) - 最终能否成交
2. **产品说明** (25%) - 专业度体现
3. **异议处理** (25%) - 解决问题的能力
4. **需求挖掘** (20%) - 了解客户的能力

## 📝 提示词特点

所有角色的提示词都包含以下完整结构:

1. **性格特征** - 描述客户的基本性格
2. **背景情况** - 年龄、职业、收入等
3. **对话特点** - 沟通风格和习惯
4. **异议类型** - 常见的反对意见
5. **评分重点** - AI评估的重点方向

这种设计确保AI能够扮演真实、有挑战性的客户,为销售训练提供有效场景。
